📐 题目

如图，在正四棱锥 $P\text{-}ABCD$ 中，底面 $ABCD$ 是边长为 $4$ 的正方形，侧棱 $PA = 2\sqrt{5}$。

点 $E$ 是 $PC$ 的中点，求直线 $BE$ 与底面 $ABCD$ 所成角的正弦值。

✓ 答案
$\sin\theta = \dfrac{\sqrt{39}}{13}$
🎯 知识点
线面角 空间向量 正四棱锥性质 建系法
💡 解题思路

以底面中心 $O$ 建立空间直角坐标系，求出各关键点坐标，利用向量法计算线面角。 线面角 $\theta$ 满足：

$$\sin\theta = \frac{|\vec{BE} \cdot \vec{n}|}{|\vec{BE}||\vec{n}|}$$
其中 $\vec{n}$ 为底面法向量
1
建立空间直角坐标系

以底面正方形中心 $O$ 为原点，$x$ 轴平行于 $AB$，$y$ 轴平行于 $BC$，$z$ 轴沿 $OP$ 方向，建立空间直角坐标系 $O\text{-}xyz$。

✦ 正四棱锥顶点 $P$ 在底面的射影恰好是底面中心 $O$
2
确定各顶点坐标

底面边长为 $4$，中心 $O$ 到各顶点距离为 $\dfrac{4\sqrt{2}}{2} = 2\sqrt{2}$

A(2, −2, 0)  B(2, 2, 0)
C(−2, 2, 0)  D(−2, −2, 0)

设 $P(0,0,h)$，由 $PA = 2\sqrt{5}$：

$$\sqrt{4+4+h^2} = 2\sqrt{5} \Rightarrow h^2 = 12 \Rightarrow h = 2\sqrt{3}$$
P(0, 0, 2√3)

$E$ 是 $PC$ 中点：

$$E = \frac{P+C}{2} = \left(\frac{0+(-2)}{2},\, \frac{0+2}{2},\, \frac{2\sqrt{3}+0}{2}\right) = (-1,\, 1,\, \sqrt{3})$$
3
求方向向量 $\vec{BE}$

$B(2,2,0)$，$E(-1,1,\sqrt{3})$

$$\vec{BE} = E - B = (-3,\; -1,\; \sqrt{3})$$

底面 $ABCD$ 的法向量：

$$\vec{n} = (0,\; 0,\; 1)$$
4
计算线面角

设直线 $BE$ 与底面所成角为 $\theta$，则：

$$\sin\theta = \frac{|\vec{BE} \cdot \vec{n}|}{|\vec{BE}| \cdot |\vec{n}|}$$

计算各量：

$$\vec{BE} \cdot \vec{n} = (-3)(0)+(-1)(0)+\sqrt{3}(1) = \sqrt{3}$$ $$|\vec{BE}| = \sqrt{(-3)^2+(-1)^2+(\sqrt{3})^2} = \sqrt{9+1+3} = \sqrt{13}$$
$$\sin\theta = \frac{\sqrt{3}}{\sqrt{13}} = \frac{\sqrt{3}\cdot\sqrt{13}}{13} = \boxed{\dfrac{\sqrt{39}}{13}}$$
以下是本题常见错误，请仔细核对：
⚠
错误一：顶点投影位置
误以为 $P$ 在底面的射影是某个顶点。正确：正四棱锥顶点 $P$ 在底面的射影是底面中心 $O$，即 $PO \perp$ 底面。
⚠
错误二：计算侧棱高度
$PA^2 = PO^2 + OA^2$，其中 $OA$ 是底面中心到顶点的距离，即对角线之半 $= \dfrac{4\sqrt{2}}{2} = 2\sqrt{2}$，所以 $OA^2 = 8$，而非 $OA = 4$。
⚠
错误三：线面角公式
线面角用正弦，而非余弦。公式为： $\sin\theta = \dfrac{|\vec{l}\cdot\vec{n}|}{|\vec{l}||\vec{n}|}$， 其中 $\vec{n}$ 是底面法向量，$\vec{l}$ 是直线方向向量。
⚠
错误四：中点坐标计算
$E$ 是 $PC$ 中点，坐标为 $\dfrac{P+C}{2}$，注意三个分量都要取平均，不要遗漏 $z$ 分量。
⚠
错误五：化简根式
$\dfrac{\sqrt{3}}{\sqrt{13}} \neq \dfrac{\sqrt{3}}{13}$，正确化简： $\dfrac{\sqrt{3}}{\sqrt{13}} = \dfrac{\sqrt{3}\cdot\sqrt{13}}{13} = \dfrac{\sqrt{39}}{13}$
🔄 变式题

在同一正四棱锥 $P\text{-}ABCD$（底面边长为 $4$，$PA = 2\sqrt{5}$）中，若点 $F$ 是 $PC$ 靠近 $P$ 的三等分点（即 $PF:FC = 1:2$），求直线 $AF$ 与底面 $ABCD$ 所成角的余弦值。

✓
变式题解析提示

$F$ 是 $PC$ 靠近 $P$ 的三等分点，即 $F = P + \dfrac{1}{3}(C-P) = \dfrac{2P+C}{3}$

$$F = \left(\frac{2\cdot0+(-2)}{3},\, \frac{2\cdot0+2}{3},\, \frac{2\cdot2\sqrt{3}+0}{3}\right) = \left(-\frac{2}{3},\, \frac{2}{3},\, \frac{4\sqrt{3}}{3}\right)$$

$A(2,-2,0)$，方向向量 $\vec{AF} = F - A = \left(-\dfrac{8}{3}, \dfrac{8}{3}, \dfrac{4\sqrt{3}}{3}\right)$

$$\sin\theta = \frac{|AF_z|}{|AF|} = \frac{\frac{4\sqrt{3}}{3}}{\sqrt{\frac{64}{9}+\frac{64}{9}+\frac{48}{9}}} = \frac{\frac{4\sqrt{3}}{3}}{\frac{4\sqrt{11}}{3}} = \frac{\sqrt{3}}{\sqrt{11}} = \frac{\sqrt{33}}{11}$$
$$\cos\theta = \sqrt{1 - \frac{3}{11}} = \sqrt{\frac{8}{11}} = \frac{2\sqrt{2}}{\sqrt{11}} = \frac{2\sqrt{22}}{11}$$
