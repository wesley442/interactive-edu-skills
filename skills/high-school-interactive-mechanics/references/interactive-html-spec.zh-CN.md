# 高中互动力学 HTML 生成规范

当用户要求“HTML”“互动页面”“动画”“变量可调”“滑块调参”时，必须生成一个完整自包含 HTML 文件。目标不是只给答案，而是把一道力学题讲清楚、画清楚、演示清楚。

## 适用场景

- 生成一道高中力学题。
- 生成斜面、摩擦力、平抛、圆周运动、连接体、弹簧、动能定理、机械能守恒、动量守恒题。
- 给已有物理题生成受力分析和分步解析。
- 生成可视化讲解、课堂演示、动画分镜。
- 生成 HTML 互动题页。
- 生成可以动态调整变量的物理模拟页面。
- 根据难度、知识点、题型生成变式题。

## 默认规则

- 如果用户没有指定年级，默认高中。
- 如果用户没有指定难度，默认中等难度。
- 如果用户没有指定题型，优先生成斜面、连接体、平抛或圆周运动题。
- 如果用户没有指定输出形式，默认输出“题目 + 题目定位 + 物理模型 + 受力分析 + 分步解析 + 答案 + 可视化讲解 + 易错点 + 变式题”。
- 如果用户要求“HTML”“互动页面”“动画”“变量可调”“滑块调参”，必须输出完整 HTML 代码。

## 生成题目前必须判断

1. 研究对象是什么？
2. 运动过程是什么？
3. 已知量是什么？
4. 未知量是什么？
5. 有哪些约束条件？
6. 需要画哪些力？
7. 应该用牛顿第二定律、运动学公式、能量法还是动量法？
8. 变量是否适合动态调整？
9. 是否存在静止、临界、反向运动或无解情况？

## HTML 页面必须满足

1. 使用原生 HTML/CSS/JS，可直接在浏览器打开。
2. 左侧显示题目、变量滑块、实时计算结果和公式推导。
3. 右侧显示物理示意图或动画。
4. 变量改变时，实时更新：
   - 题目参数
   - 计算结果
   - 受力箭头
   - 运动状态
   - 关键结论
5. 使用 MathJax 渲染公式。
6. 优先使用 SVG 或 Canvas 绘制 2D 物理图。
7. 动画使用 `requestAnimationFrame`。
8. 不要只做静态图，必须有可拖动变量控件。
9. 必须限制变量范围，避免出现不合理或无法解释的状态。
10. 当物体不会运动、临界状态、方向改变或结果为负时，必须在页面中明确提示。
11. 页面应适合课堂投屏，文字清晰，布局简洁。

## 推荐互动变量

### 斜面摩擦力

可调变量：

- 质量 m
- 斜面角 θ
- 动摩擦因数 μ
- 重力加速度 g

实时计算：

- G = mg
- N = mg cosθ
- f = μN
- F = mg sinθ - μmg cosθ
- a = g(sinθ - μcosθ)

可视化：

- 斜面角度随 θ 改变。
- 重力竖直向下。
- 支持力垂直斜面。
- 摩擦力沿斜面向上。
- 合力沿斜面方向。
- 小物块根据加速度运动。

### 平抛运动

可调变量：

- 初速度 v₀
- 高度 h
- 重力加速度 g

实时计算：

- t = sqrt(2h/g)
- x = v₀t
- vy = gt
- v = sqrt(v₀² + vy²)

可视化：

- 轨迹抛物线。
- 水平速度箭头。
- 竖直速度箭头。
- 合速度箭头。
- 落地点距离实时变化。

### 圆周运动

可调变量：

- 质量 m
- 半径 r
- 速度 v

实时计算：

- a = v²/r
- F = mv²/r
- T = 2πr/v

可视化：

- 物体绕圆运动。
- 速度箭头沿切线。
- 向心加速度指向圆心。
- 合力指向圆心。

### 弹簧问题

可调变量：

- 弹簧劲度系数 k
- 质量 m
- 压缩量 x

实时计算：

- 弹性势能 Ep = 1/2kx²
- 最大速度 v = x sqrt(k/m)
- 弹力 F = kx

可视化：

- 弹簧伸缩。
- 弹力箭头变化。
- 能量柱状图变化。

### 连接体

可调变量：

- m₁
- m₂
- 摩擦因数 μ
- 外力 F

实时计算：

- 系统加速度。
- 绳子拉力。
- 各物体受力。

可视化：

- 两个物体同步运动。
- 拉力箭头。
- 摩擦力箭头。
- 整体法与隔离法切换。

## HTML 页面结构建议

- 顶部：标题和题目摘要。
- 左侧面板：
  - 题目
  - 变量滑块
  - 实时结果
  - 公式推导
  - 状态提示
- 右侧面板：
  - SVG/Canvas 动画
  - 受力箭头
  - 速度/加速度方向
  - 播放/暂停/重置按钮
- 底部：
  - 易错点
  - 课堂提问
  - 变式题

## 质量要求

- 不要只写公式，要解释为什么这样列方程。
- 受力分析必须明确研究对象。
- 摩擦力方向必须根据相对运动或相对运动趋势判断。
- 加速度方向、速度方向、合力方向不能混淆。
- 向心力不是一种额外的新力，必须说明它是合力或某个具体力提供的效果。
- 计算结果必须和推导过程一致。
- 公式、单位、数值范围必须自检。
- 输出语言要像优秀高中物理老师讲题，清楚、严谨、适合课堂使用。
