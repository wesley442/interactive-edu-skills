# YouMind Form: High-School Interactive Mechanics

## 技能名称

高中互动力学题工坊

## 副标题

生成可调变量的受力图、运动动画与分步解析

## 描述

面向高中物理老师、家教和学生的力学题生成与互动讲解工具。输入知识点、题型或一句需求，即可生成完整力学题、受力分析、公式推导、标准答案、易错点和变式题；也可以生成可直接打开的 HTML 互动页面，通过滑块动态调整变量，实时更新受力图、运动动画和计算结果。

## 展示提示词

1. 生成一道高中斜面摩擦力互动题页，可以调整质量、斜面角度和摩擦因数。
2. 生成一个平抛运动 HTML 动画页面，可以调整初速度和高度。
3. 生成一道连接体力学题，要求有整体法、隔离法和可调变量动画。
4. 生成一道圆周运动题，讲清楚向心力来源，并做成可调速度和半径的互动页面。
