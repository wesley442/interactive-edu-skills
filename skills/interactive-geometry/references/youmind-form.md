# YouMind Form: Interactive Geometry

## 技能名称

互动几何题工坊

## 副标题

生成可视化几何题、分步解析与互动教学页

## 描述

把几何题从“看解析”升级成“能看见”。输入知识点、年级、难度或直接说“随机出一道题”，即可生成一套完整几何题：题干、答案、分步解析、易错点、变式题和可视化讲解设计。也可生成可直接打开的 HTML 互动教学页，适合数学老师备课、家教出题、学生自学复盘和课堂演示。

## 展示提示词

1. 随机生成一道高中立体几何线面角题，并给出互动教学页设计。
2. 出一道正方体中的异面直线夹角题，难度中等，适合课堂讲解。
3. 生成一道点到平面距离题，要求答案整洁，并提供完整解析。
4. 根据这张几何题图片，识别题目并生成分步解析和可视化讲解。
